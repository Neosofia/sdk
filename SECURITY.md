# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in any package in
this repository, please report it privately.

**Email:** security@neosofia.com

Please include:
- The package and version (e.g. `neosofia-logger 0.1.0`)
- A description of the issue and its potential impact
- A reproduction (minimal code that demonstrates the problem)

We will acknowledge receipt within 5 business days and aim to ship a
patched release within 30 days of confirmation.
